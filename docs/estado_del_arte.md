# Referencias técnicas esenciales

La base teórica del repositorio debe apoyarse prioritariamente en fuentes
primarias y documentación oficial:

1. Chen, T. y Guestrin, C. *XGBoost: A Scalable Tree Boosting System* (KDD 2016).
2. Documentación oficial de XGBoost sobre parámetros, objetivos y predicción.
3. CDC/NCHS, documentación de variables y ciclos de NHANES.
4. Documentación oficial de PyCaret 3 para experimentos de clasificación.
5. Documentación de scikit-learn para particiones, calibración y métricas.

Los artículos divulgativos pueden utilizarse para intuición, pero no deben
sustituir la definición oficial de variables NHANES ni la metodología de
evaluación.

## Distinción metodológica

El proyecto no debe compararse directamente con estudios prospectivos de riesgo
cardiovascular: su objetivo histórico es prevalencia autorreportada de
antecedente de infarto. Una comparación válida exige equivalencia en población,
desenlace, horizonte temporal, variables y estrategia de validación.
